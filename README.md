# migri
A super simple PostgreSQL migration tool that uses asyncpg.

## Getting started
### Install `migri`
Run `pip install https://github.com/RonquilloAeon/migri/archive/master.zip`

### Initialize
Run `migri init` to create the table that tracks migrations.
